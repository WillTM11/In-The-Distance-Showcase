# ITD Clothing Store

Live site: [(https://inthedistanceitd.com/)](https://inthedistanceitd.com/)

A full-stack e-commerce store built with:
- HTML, CSS, JavaScript
- Node.js + Express
- MySQL
- Stripe Checkout + Webhooks
- Render deployment

## Features
- Product management
- Admin dashboard
- Order tracking
- Stock management
- Stripe payments
- Responsive UI
